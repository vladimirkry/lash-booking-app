# Lash Atelier

Фронтенд MVP для онлайн-запису на нарощення вій з окремими входами для клієнта та адміністратора.

## Входи

- `client.html` — публічна клієнтська форма для QR-коду та посилання
- `admin.html` — окремий вхід адміністратора
- `index.html` — поточна робоча оболонка з маршрутом `#client`, `#admin` або `#share`

Клієнтський вхід відкриває тільки форму запису. Адмінський вхід відкриває кабінет і QR/посилання, але не дає переходу в клієнтський режим через меню.

У цій локальній версії PIN-коду немає. Для публічного запуску потрібна справжня авторизація адміністратора через backend, наприклад Supabase Auth або Firebase Auth.

## Що вже працює

- Вибір мови: українська, польська, нідерландська, англійська
- Послуги: нарощення вій 2D, 3D, 4D, 5D
- Клієнт бачить тільки доступні дні та години
- Адміністратор керує робочими днями й годинами прийому
- Адміністратор додає вихідні / закриті дати
- Записи враховують тривалість послуги та не накладаються
- Адміністратор створює, підтверджує, завершує, скасовує та видаляє записи
- Клієнт не змінює запис напряму, а надсилає заявку
- Адміністратор бачить заявки клієнтів
- QR-код для публічного клієнтського посилання
- PWA manifest, іконка та кешування після публікації

## Що потрібно для повноцінного сайту

- Хостинг: Netlify, Vercel, GitHub Pages або VPS
- Домен, наприклад `booking.your-salon.com`
- Backend і база даних: Supabase або Firebase
- Авторизація адміністратора
- Таблиці для записів, графіка, вихідних, послуг і заявок
- Серверні сповіщення адміністратору
- Політика приватності для контактних даних клієнтів

## Рекомендований наступний етап

Найкращий шлях для цього проєкту: Supabase для бази, авторизації й realtime-оновлень, плюс email або Telegram-сповіщення на старті. SMS/WhatsApp краще додавати пізніше, коли вже зрозумілий бюджет і країна основних клієнтів.
